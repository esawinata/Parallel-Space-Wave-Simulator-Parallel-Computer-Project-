Evaluasi 3 - IFB 206 Komputasi Paralel

Parallel Space Wave Simulator

Simulasi game top-down shooter menggunakan C# dengan perbandingan serial vs parallel processing.

Deskripsi Project

Parallel Space Wave Simulator adalah prototype game sederhana bertema space shooter. Program menampilkan player, enemy, dan projectile sebagai visualisasi, lalu membandingkan performa update enemy dalam mode serial dan paralel.

Konsep Komputasi Paralel

Konsep yang digunakan adalah Data Parallelism. Banyak enemy dalam game memiliki data yang dapat diproses secara independen, seperti posisi, HP, defense, dan status hidup/mati. Karena update satu enemy tidak bergantung pada enemy lain, proses tersebut dapat diparalelkan menggunakan Parallel.For pada C#.

Mode Serial

Enemy diproses satu per satu menggunakan for-loop.

Mode Paralel

Enemy dibagi ke beberapa thread CPU menggunakan Parallel.For.

Alur Program

  1. Program memuat sprite Player, Enemy, dan Projectile dari folder Assets.
  2. Game menampilkan visual top-down shooter sederhana.
  3. User menekan tombol B untuk menjalankan benchmark.
  4. Program menjalankan simulasi update enemy secara serial.
  5. Program menjalankan simulasi update enemy secara paralel.
  6. Program menampilkan waktu eksekusi dan speedup.

Cara Menjalankan

dotnet run

Kontrol: WASD/Arrow untuk bergerak, tahan Space untuk menembak terus-menerus, B untuk benchmark serial vs parallel.

Kesimpulan

Project ini menunjukkan bahwa proses update banyak enemy dalam game dapat dipercepat dengan komputasi paralel, selama data setiap enemy dapat diproses secara independen.

Runtime Mode

Tekan Z untuk Serial Runtime Mode dan X untuk Parallel Runtime Mode. Overlay game menampilkan FPS, frame time, runtime simulation time, jumlah enemy runtime, serta hasil benchmark Serial Time, Parallel Time, dan Speedup.
